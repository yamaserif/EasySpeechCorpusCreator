【概要】
    本ソフトは音声コーパスの収録を、簡単に行うためのツールです。
    ソフト上で原稿を確認しながら、収録や音声データの確認ができます。
    また、コーパスに関する情報の追加、管理ができます。

● ソフトの使い方
・管理画面
コーパスに関する情報の管理ができます。

No. : 管理番号
名前 : 管理名
文章 : 読み上げる原稿
カナ : 読み上げる原稿（カタカナ）
タグ : 自由に設定できるメタデータ

基本的には設定したい数値文字を入力できますが、タグの入力は特殊な形式になっています。
タグは情報を区切って複数入力でき、]を入力することで新たにデータを区切ることができます。

また、この画面では音声の収録が可能です。
スペースキーを押している間は録音状態となり、音声が保存されるとその波形が表示されます。
zキーにて収録音声を聞くこともできます。


・インポート画面
テキストデータとして公開されているコーパスの原稿ファイルを取り込むことができます。
インポートする際はそのデータの形式を設定することで、様々なフォーマットのファイルに対応できます。

%Name% : 管理名
%Sentence% : 読み上げる原稿
%Kana% : 読み上げる原稿（カタカナ）
%_Invalid% : 無視するデータ※

※実は内部的には「_Invalid」である必要はなく、その他の置換用の変数名に被らなければ何でも良かったりします。


・設定画面
各種設定ができます。
基本的には説明にある通りの設定です。
設定後は保存ボタンを忘れずクリックしてください。


・収録したデータについて
projectsフォルダにデータが保存されます。
コーパスのプロジェクト毎にフォルダが作成されるため、そのままデータを利用・配布することが可能です。
また、管理画面で設定した各データはJSONやCSVに書き出されます。
収録したコーパスをプログラムで処理したい場合に役立つと思います。
